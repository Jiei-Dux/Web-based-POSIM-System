Help Page

Introduction

Meogeo Sarang Korean Mart | Point-of-Sales and Inventory Management System. This comprehensive solution is designed to streamline your business operations, enhance customer service, and provide valuable insights into your sales and inventory

Purpose and Benefits

  • Efficient sales processing
  • Real-time inventory tracking
  • Customer relationship management
  • Data-driven decision making through reports and analytics

Getting Started

Logging In

  1. Acquire your Employee ID and password from your manager
  2. Visit the Web application
  3. Enter your Employee ID and password
  4. Click "Login" or press ENTER

Initial Setup

For initial setup and configuration, please contact your system administrator to set up user profiles and permissions.

Navigation Guide

The main interface consists of several key areas:

  • Left Navigation Bar: Quick access to main features [ Home, Stocks, Reports, Settings ]
  • Top Bar: Search bar, Database Connectivity Refresher and Checker
  • Main Content Area: Where you'll perform most operations
  • Right Shopping Cart: Where the selected products will appear

Basic Operations

  1. Either manually navigate and scroll through the list of products or use the search bar
  2. Scan items or Add them manually
  3. Apply any discounts if necessary
  4. Ask if the customer wanted to Pay-Later or Proceed to Payment
  5. Select payment method
  6. Complete the transaction

Access the Inventory module through the Stocks Tab to add new items, update existing item details, and track inventory levels.

Advanced Features

Reports and Analytics

Access the Reports module to generate and view various sales and inventory reports.

Employee Management

Administrators can access the Employees module to manage profiles, set permissions, and track performance.

Troubleshooting

Common Issues

  • System not responding: Try refreshing the page, restarting the browser, or check for internet connectivity
  • Transaction errors: Double-check all entered information and try processing again
